The last time that we sent a message to the irc network, used to prevent flooding.

Sends a message to the irc network

Note: the message is not directly sent, it will be cached locally to prevent flooding See Also: flood_interval

Property

self - command - the string to send, may contain %(*)s wich will be replaced with ... callback - the function to call when the send is completed ...  - arguments for %{}
