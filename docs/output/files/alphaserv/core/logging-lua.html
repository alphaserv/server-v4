Internal util to format a name from a cn.

Format

  • name
  • cn
  • ip
  • country

Parameters

Return

The formated String.

Example Return

debuglevels

debuglevels

An array containing the debuglevel ids that are available to log to.

Note: these are alos available under the global scope prefixed with “LOG_”

FATALUnrecoverable error
ERRORUnexpected behaviour that should not happen
NOTICEErrors that may cause bugs but can usally be ignored
INFOMessages that are usefull when debugging

debug

function debug(level,
text)

Logs messages that are selected to be logged to log/debug.log

Parameters

levelThe level to log to,
textThe formated text

Possible Values for level

  • LOG_FATAL note: this also trows up an error
  • LOG_ERROR
  • LOG_NOTICE
  • LOG_INFO

Note

This function is also available in the global scope named “log_msg”
