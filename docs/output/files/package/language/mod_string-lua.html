function _str_interp(s,
t)

String interpolation function taken and modified from here http://lua-users.org/wiki/StringInterpolation.  Unlike the way in the link, we also handle unnamed arguments, _ and numbers.

It is activated using

getmetatable("").__mod = _str_interp

Parameters

sThe input string.
tA table of arguments to substitute.

Returns

Final string with all substitutions done.

split

function split(s,
d)

Splits a string into table of tokens, based on http://lua-users.org/wiki/SplitJoin.  Usage -

local a = "abc|def|ghi|jkl"
local b = string.split(a, '|')
assert(table.concat(b) == "abcdefghijkl")

Parameters

sThe string to split.
dA delimiter to use.

Returns

A table of tokens.

template

function template(s,
l,
e)

Parses a string template (string with embedded lua code), inspired by luadoc parser system.  Usage -

foo = "bar"
bar = "blah"
-- this returns "bar: blah"
-- first, gets parsed to "bar : <$0 return bar $0>"
-- then, it gets parsed to "bar : blah" (value of bar)
assert(
    string.template("bar : <$0 return <$1=foo$1> $0>") == "bar : blah"
)

Parameters

sThe string to parse.
lLevel to parse string from.  Everything with higer or equal level gets parsed.  Defaults to 0.
eoptional associative table of items that should be visible to the embedded lua code as part of environment.

Returns

Parsed string.
