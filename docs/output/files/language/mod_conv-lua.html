Purpose

This file implements a conversion module for OctaForge Lua API.  It features type conversion functions and color converion functions (RGB/HSV/HSL/hex).

convert

A conversion library for OctaForge Lua API.  Contains several basic type conversion methods, some of them globally-wrapped, and some color conversion functions.
