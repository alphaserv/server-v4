Commits the current transaction.  This feature might not work on database systems that do not implement transactions.  Returns: true in case of success and false when the operation could not be performed or when it is not implemented.

setautocommit

setautocommit = function(self,
value,
...)

Function: setAutoCommit Turns on or off the “auto commit” mode.  This feature might not work on database systems that do not implement transactions.  On database systems that do not have the concept of “auto commit mode”, but do implement transactions, this mechanism is implemented by the driver.  Returns: true in case of success and false when the operation could not be performed or when it is not implemented.
